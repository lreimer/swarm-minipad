# Docker Swarm Minipad

Visualise and manage Docker swarm containers using a Novation Launchpad Mini MK2.

## Building and Running

```bash
$ ./gradlew clean ass
$ ./gradlew run --args="-f src/test/resources/swarm-config.json"
```
